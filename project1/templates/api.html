API Usage

Sending a GET request to the following URL http://127.0.0.1:5000/api/[ISBN], where [ISBN] should be changed by the real identificational number of the book, you will get a JSON in response.

Request example: http://127.0.0.1:5000/api/1857984986

JSON Model example:
{
"title": "The Stainless Steel Rat",
"author": "Harry Harrison",
"year": "1961",
"isbn": "1857984986",
"review_count": 12125,
"average_score": "3.93"
}
